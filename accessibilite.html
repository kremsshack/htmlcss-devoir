Accessibilité

Les lecteurs d’écran ont besoin d’une structure HTML sémantique pour naviguer correctement.

Les liens doivent indiquer clairement où ils mènent pour éviter les erreurs.

Les boutons doivent être accessibles au clavier via la touche Tabulation.

Une interface accessible réduit les risques d’erreurs involontaires.

Limiter les animations évite de gêner les utilisateurs sensibles ou neuroatypiques.
